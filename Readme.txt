Mit diesem kleinen Batch-Script (GT_backup.bat) kann auf einfachste Weise ein inkrementelles Backup von beliebig vielen Ordnern erstellt werden.
Dafür müssen die folgenden Schritte nacheinander ausgeführt werden:

1) ganzen Ordner auf die Festplatte kopieren (zum Beispiel C:\Temp)
2) install_GT_backup.bat anklicken
3) Verknüpfung mit GT_backup.bat auf den Desktop erstellen
4) In der Datei bkpfolder.txt die Pfade für die Datensicherung angeben. (jeweils auf einer neuen Zeile!)


Von nun an kann per doppelklick auf das Icon ein Backup der angegebenen Ordner erstellt werden.
Die Backups werden automatisch im Home-Verzeichnius des jeweiligen Benutzers unter bkp_develo gespeichert.
Dateien die bereits einmal kopiert wurden und seither nicht geändert haben, werden nicht erneut kopiert!
